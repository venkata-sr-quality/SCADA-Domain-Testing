# SCADA System – Senior Test Engineering Portfolio

This repository represents how I approach testing of
SCADA, industrial control, and cooling infrastructure systems
from a **Senior Test Engineer and domain expert perspective**.

This is NOT a client project repository.

The focus here is on:
- System-level testing
- Control logic validation
- Communication reliability
- Safety, stability, and long-run behavior
- Release readiness in industrial systems

The content is derived from real-world experience in:
- SCADA and device drivers
- Gateways (Serial, Ethernet, Serial–Ethernet)
- Direct Liquid Cooling (DLC) systems
- Long-run, stress, and failure testing
- QA leadership and release quality
